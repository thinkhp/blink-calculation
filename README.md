# 眨眼计数器

#### install
```
conda create -n blinkcalculation python=3.9
conda activate blinkcalculation

conda install numpy scipy
conda install -c conda-forge opencv dlib

# mac
python ./blink_calculation_mac.py
# win
python ./blink_calculation_win.py
```
